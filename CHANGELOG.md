# Changelog

All notable changes to `acl-arr-paper-drafter` will be documented in this file.

## v0.2.1 - 2026-04-12

- Added an ACL / EMNLP empirical paper-story pattern centered on blind spots, benchmark design, systematic evaluation, and proportional mitigation
- Added reusable title, abstract, introduction, contribution, and experiment-writing templates to `references/research-writing-patterns.md`
- Updated the main skill instructions to explicitly apply these patterns during ACL ARR style drafting

## v0.2.0 - 2026-03-29

- Strengthened the drafting workflow with patterns distilled from the local research-writing skill library
- Added `references/research-writing-patterns.md`
- Added claim-evidence mapping and citation-plan requirements to the main drafting flow
- Added a problem -> solution -> evidence narrative check before prose drafting
- Added section-level quality checks to reduce generic or overclaimed writing
- Kept the original XMind-derived paper framework while improving research-writing discipline
- Refreshed the public repository README for clearer installation, usage, and positioning

## v0.1.0 - 2026-03-29

- Initial public release
- Supports ACL ARR style LaTeX draft generation from topic plus mixed materials
- Includes official ACL-style and ARR submission guidance
- Includes a minimal compiled example for `Personality in LLM agents`
